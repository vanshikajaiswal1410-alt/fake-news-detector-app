# 🧠 AI Fake News Detector App

A Flutter-based AI-powered mobile app that detects whether a news headline or URL is Fake or Real.

## 🚀 Features

- 🧠 AI News Detection (Fake / Real)
- 
- 🔗 URL-based News Analysis
- 
- 🎤 Voice Input (Speech-to-Text)
- 
- 📰 Live News (India Headlines)
- 
- 📱 Clean & User-Friendly UI

## 🛠️ Tech Stack

- Flutter
- Dart
- OpenRouter API (AI)
- News API
- Speech-to-Text
